Layout e Posicionamento

Layout em CSS define como os elementos são organizados no espaço da tela. Evoluímos de fluxos de texto simples para sistemas complexos de duas dimensões (Grid) e distribuições flexíveis (Flexbox).

display — O comportamento do elemento

Define o tipo de "caixa" que o elemento gera. É a propriedade mais importante para o fluxo do documento. 

div {
  display: block;
}

Valores Principais

  • block — Ocupa toda a largura, começa em nova linha (ex: div, p, h1).
  • inline — Ocupa apenas o tamanho do conteúdo, não aceita width/height (ex: span, a).
  • inline-block — Flui como texto, mas aceita dimensões de bloco.
  • none — Remove o elemento e seu espaço do documento.
  • flex — Transforma o elemento em um flex-container.
  • grid — Transforma o elemento em um grid-container.

position — Posicionamento Físico

Define como um elemento é posicionado em relação ao fluxo, ao pai ou à tela. 

.box {
  position: absolute;
  top: 20px;
  right: 10px;
}

Valores e Comportamentos

  • static — Padrão. Segue o fluxo natural da página.
  • relative — Posicionado em relação a **si mesmo**. O espaço original é preservado.
  • absolute — Posicionado em relação ao **ancestral posicionado** mais próximo (qualquer um que não seja static). Sai do fluxo.
  • fixed — Posicionado em relação à **janela do navegador (viewport)**. Não move no scroll.
  • sticky — Alterna entre relative e fixed dependendo do scroll.

Propriedades de Auxílio

Usadas apenas quando position não é static:

  • top, bottom, left, right
  • z-index — Controla a pilha no eixo Z (profundidade).

Flexbox — Container (Pai)

Layout unidimensional (linha ou coluna). Foca na distribuição de espaço e alinhamento. 

.container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
}

Propriedades do Pai

  • flex-direction — Direção: row, column, row-reverse, column-reverse.
  • flex-wrap — Quebra de linha: nowrap, wrap.
  • justify-content — Alinhamento no **eixo principal**: flex-start, center, space-between, space-around, space-evenly.
  • align-items — Alinhamento no **eixo transversal**: stretch, center, flex-start, flex-end.
  • gap — Espaçamento entre os itens flex.

Flexbox — Items (Filhos)

Controlam como cada item individual se comporta dentro do flex-container.

  • flex-grow — Capacidade de crescer (0 = não cresce).
  • flex-shrink — Capacidade de encolher.
  • flex-basis — Tamanho base antes da distribuição de espaço.
  • align-self — Sobrescreve o align-items para um item específico.
  • order — Altera a ordem visual de exibição.

CSS Grid Layout

Layout bidimensional (linhas e colunas simultâneas). O sistema mais poderoso do CSS. 

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3 colunas iguais */
  grid-template-rows: auto;
  gap: 20px;
}

Conceitos de Grid

  • grid-template-columns — Define colunas (ex: 100px 1fr 20%).
  • grid-template-areas — Cria layouts nomeando áreas.
  • fr — Unidade de fração (ocupa o espaço disponível).
  • repeat(3, 1fr) — Atalho para 3 colunas iguais.

Posicionamento no Grid (Filhos)

.item {
  grid-column: 1 / 3; /* Ocupa da linha 1 até a 3 */
  grid-row: 2;
}

Float e Clear (Legado/Específico)

Antigamente usado para layout, hoje serve principalmente para envolver texto em volta de imagens. 

img { float: left; margin: 10px; }
.limpeza { clear: both; }

Resumo de Escolha

  • Position: Use para sobreposições, menus fixos ou elementos isolados.
  • Flexbox: Use para alinhar elementos em uma direção (menus, barras laterais, centralização).
  • Grid: Use para a estrutura geral da página ou layouts complexos de galeria.
