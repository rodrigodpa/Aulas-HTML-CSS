Texto e Tipografia em CSS

CSS é responsável por controlar a aparência do texto: fontes, tamanhos, alinhamento, espaçamento e decoração. Tipografia bem feita melhora leitura, hierarquia visual e usabilidade.

font-family — Família da fonte

Define a fonte usada no texto. Sempre forneça fontes alternativas. 

p {
  font-family: Arial, Helvetica, sans-serif;
}

Use fontes genéricas como fallback:

  • serif
  • sans-serif
  • monospace
  • cursive
  • fantasy

font-size — Tamanho da fonte

Controla o tamanho do texto. 

p {
  font-size: 16px;
}

Unidades comuns

  • px — pixels (fixo)
  • em — relativo ao elemento pai
  • rem — relativo ao body
  • % — porcentagem

Recomendação: usar rem.

font-weight — Peso da fonte

Define a espessura do texto. 

p {
  font-weight: bold;
}

Valores possíveis

  • normal
  • bold
  • lighter
  • bolder
  • 100–900

font-style — Estilo da fonte

em {
  font-style: italic;
}
  • normal
  • italic
  • oblique

font — Shorthand

Define várias propriedades de fonte em uma linha. 

p {
  font: italic bold 1.2rem Arial, sans-serif;
}

Ordem obrigatória:

  1. font-style
  2. font-weight
  3. font-size
  4. font-family

line-height — Altura da linha

Controla o espaçamento vertical entre linhas. 

p {
  line-height: 1.6;
}

Dica: valores sem unidade são recomendados.

text-align — Alinhamento do texto

p {
  text-align: justify;
}
  • left
  • right
  • center
  • justify

text-indent — Recuo da primeira linha

p {
  text-indent: 2em;
}

Muito usado em textos longos e artigos.

letter-spacing — Espaçamento entre letras

h1 {
  letter-spacing: 2px;
}

Use com moderação. Valores altos prejudicam leitura.

word-spacing — Espaçamento entre palavras

p {
  word-spacing: 4px;
}

text-transform — Transformação do texto

p {
  text-transform: uppercase;
}
  • uppercase
  • lowercase
  • capitalize

Não altera o conteúdo real, apenas a exibição.

text-decoration — Decoração do texto

a {
  text-decoration: none;
}

Propriedade composta:

  • text-decoration-line
  • text-decoration-style
  • text-decoration-color

white-space — Tratamento de espaços

pre {
  white-space: pre-wrap;
}
  • normal
  • nowrap
  • pre
  • pre-wrap
  • pre-line

text-overflow — Texto excedente

p {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}

Usado para cortar texto com reticências.

color — Cor do texto

p {
  color: #333;
}

Aceita:

  • nomes (red)
  • hex (#ff0000)
  • rgb()
  • hsl()

text-shadow — Sombra no texto

h1 {
  text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
}

Ordem:

  • deslocamento X
  • deslocamento Y
  • desfoque
  • cor

Resumo

  • Tipografia controla legibilidade
  • Use unidades relativas
  • Prefira semântica no HTML
  • CSS apenas estiliza
