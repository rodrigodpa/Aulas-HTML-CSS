Box Model — Modelo de Caixas

No CSS, cada elemento é representado como uma caixa retangular. O Box Model descreve as camadas que compõem essa caixa: conteúdo, preenchimento, borda e margem.

As Camadas do Box Model

Do centro para fora, as camadas são:

  • Content (Conteúdo): Onde o texto e as imagens aparecem.
  • Padding (Preenchimento): Espaço interno entre o conteúdo e a borda.
  • Border (Borda): Uma linha que envolve o preenchimento e o conteúdo.
  • Margin (Margem): Espaço externo que separa o elemento de outros elementos.

Width e Height — Largura e Altura

Definem as dimensões da área de conteúdo do elemento. 

div {
  width: 300px;
  height: 200px;
}

Variações de Limite

  • min-width / min-height — Tamanho mínimo.
  • max-width / max-height — Tamanho máximo (essencial para responsividade).

box-sizing — Cálculo do Tamanho

Define como a largura e altura totais são calculadas. Esta é uma das propriedades mais críticas do CSS. 

/* Padrão do navegador */
div { box-sizing: content-box; }

/* Recomendado para layouts modernos */
div { box-sizing: border-box; }

Valores possíveis

  • content-box: width define apenas o conteúdo. Padding e border são **somados** por fora (fazendo a caixa crescer).
  • border-box: width define o tamanho total da caixa. Padding e border são **incluídos** dentro do valor.

Dica: A maioria dos desenvolvedores usa border-box em todos os elementos.

padding — Preenchimento Interno

Cria espaço dentro do elemento. O fundo do elemento (background) é visível no padding. 

div {
  padding: 20px; /* Todos os lados */
}

Shorthand (Ordem do relógio)

  • padding: 10px; — Todos os lados.
  • padding: 10px 20px; — [Top/Bottom] [Left/Right].
  • padding: 10px 5px 15px; — [Top] [Left/Right] [Bottom].
  • padding: 10px 20px 30px 40px; — [Top] [Right] [Bottom] [Left].

margin — Margem Externa

Cria espaço fora do elemento. Margens são transparentes. 

div {
  margin: 0 auto; /* Centraliza elementos de bloco horizontalmente */
}

Margin Collapse (Colapso de Margem)

Quando dois elementos verticais se tocam, as margens não se somam; prevalece a maior.

border — Bordas

Propriedade composta para estilizar a linha em volta do elemento. 

div {
  border: 2px solid #000;
}

Subpropriedades

  • border-width: Espessura (px, thin, medium, thick).
  • border-style: Estilo (solid, dashed, dotted, double, none).
  • border-color: Cor.
  • border-radius: Arredonda os cantos da caixa.

outline — Contorno

Semelhante à borda, mas **não ocupa espaço** no Box Model e não afeta o layout. 

input:focus {
  outline: 3px solid blue;
}
  • Não é afetado por border-radius.
  • Fica do lado de fora da borda.

Comportamento de Bloco vs Linha

O Box Model se comporta de forma diferente dependendo da propriedade display:

  • Block: Respeita todas as propriedades de largura, altura, margem e padding. Ocupa toda a largura disponível.
  • Inline: width e height são ignorados. Margens e paddings laterais funcionam, mas verticais não afastam outros elementos.
  • Inline-Block: O elemento flui como texto, mas aceita todas as propriedades de tamanho do Box Model.
