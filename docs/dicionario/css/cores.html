Cores e Backgrounds em CSS

CSS utiliza diversos sistemas para definir **cores** e **fundos**. O uso correto de cores impacta diretamente a identidade visual, a acessibilidade (contraste) e a experiência do usuário.

Formatos de Cores

Existem várias formas de declarar uma cor no CSS. O navegador converte todas para um valor interno, mas cada formato é útil em cenários diferentes.

1. Color Keywords (Nomes)

Nomes pré-definidos como red, blue, transparent.

p { color: RebeccaPurple; }

2. Hexadecimal (HEX)

Baseado em 3 ou 6 dígitos (RRGGBB). Opcionalmente aceita 8 dígitos para transparência (RRGGBBAA).

h1 { color: #ff5733; }
span { color: #ff573380; } /* 50% de opacidade */

3. RGB e RGBA

Define a mistura de Red, Green e Blue (0-255). O a define o canal Alpha (transparência 0 a 1).

div { background-color: rgb(255, 87, 51); }
div { background-color: rgba(255, 87, 51, 0.5); }

4. HSL e HSLA

Hue (Matiz 0-360), Saturation (Saturação %) e Lightness (Luminosidade %). Mais intuitivo para humanos.

section { background-color: hsl(12, 100%, 60%); }

color — Cor do Texto

Define a cor da fonte e de decorações de texto.

body { color: #222222; }

background-color — Cor de Fundo

Define uma cor sólida para o fundo do elemento.

.caixa { background-color: lightgray; }

opacity — Opacidade

Define a transparência de todo o elemento (incluindo o texto dentro dele).

img { opacity: 0.7; }

Diferença crucial: opacity afeta os filhos; rgba() afeta apenas a cor específica.

linear-gradient() — Gradiente Linear

Cria uma transição suave entre duas ou mais cores em uma linha reta. É aplicado na propriedade background ou background-image.

.gradiente {
  background: linear-gradient(to right, red, yellow);
}

Direções possíveis

  • to top / to bottom
  • to left / to right
  • to bottom right (diagonais)
  • 45deg (ângulos específicos)

radial-gradient() — Gradiente Radial

O gradiente surge de um ponto central (círculo ou elipse).

.circulo {
  background: radial-gradient(circle, blue, green);
}

background-image — Imagem de Fundo

Coloca uma imagem atrás do conteúdo.

header {
  background-image: url('imagem.jpg');
}

background-repeat — Repetição

Define se a imagem de fundo deve se repetir.

  • repeat (padrão)
  • no-repeat (uma única vez)
  • repeat-x (apenas horizontal)
  • repeat-y (apenas vertical)

background-size — Tamanho do Fundo

Controla como a imagem é redimensionada.

  • auto (tamanho original)
  • cover (preenche tudo, pode cortar bordas)
  • contain (mostra a imagem inteira, pode sobrar espaço)
  • 100% 100% (estica para preencher)

background-attachment — Fixação

Define se o fundo rola com a página ou fica fixo.

  • scroll (padrão)
  • fixed (efeito de paralaxe - o fundo não se move)

background-position — Posicionamento

Define o ponto inicial da imagem de fundo.

div { background-position: center center; }
  • top, bottom, left, right, center
  • Valores em px ou %

background — Shorthand

Combina todas as propriedades de fundo em uma única linha.

div {
  background: #000 url('fundo.png') no-repeat center / cover fixed;
}

Ordem comum: color | image | repeat | position / size | attachment

Acessibilidade e Boas Práticas

  • Contraste: Garanta que o texto seja legível sobre a cor de fundo (relação mínima de 4.5:1).
  • Fallback: Ao usar gradientes ou imagens, sempre defina uma background-color sólida para caso o recurso falhe.
  • Daltonismo: Não use apenas cores para transmitir informação (ex: use ícones além do vermelho para erros).
