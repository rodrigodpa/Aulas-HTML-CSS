Links em HTML

Links são definidos pela tag <a> (âncora) e servem para conectar documentos, seções, arquivos ou ações. Eles são um dos pilares da Web.

<a> — Âncora

A tag <a> cria um link. Seu atributo essencial é href. 

<a href="https://example.com">Site externo</a>

Exemplo de link externo

Atributo href

Define o destino do link.

  • URL absoluta: https://site.com
  • URL relativa: pagina.html
  • Âncora interna: #secao
  • Email: mailto:
  • Telefone: tel:

Link interno (âncora)

Usado para navegar dentro da mesma página. 

<a href="#topo">Voltar ao topo</a>

Atributo target

Controla onde o link será aberto. 

<a href="pagina.html" target="_self">Mesmo local</a>
<a href="pagina.html" target="_blank">Nova aba</a>
  • _self — padrão
  • _blank — nova aba

Atributo rel

Define o relacionamento entre o documento atual e o destino. 

<a href="https://externo.com" target="_blank" rel="noopener noreferrer">
  Link seguro
</a>

Boa prática: sempre usar rel com _blank.

Link para download

Permite baixar arquivos diretamente. 

<a href="arquivo.pdf" download>Baixar PDF</a>

Link de email

<a href="mailto:exemplo@email.com">Enviar email</a>

Link de telefone

<a href="tel:+551199999999">Ligar</a>

Link envolvendo outros elementos

Um link pode envolver textos, imagens ou blocos inteiros. 

<a href="pagina.html">
  <img src="imagem.png" alt="Imagem">
</a>

Acessibilidade em links

  • Texto do link deve ser descritivo
  • Evitar "clique aqui"
  • Usar title apenas quando necessário
