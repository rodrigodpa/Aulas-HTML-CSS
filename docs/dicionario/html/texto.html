Texto e Semântica no HTML

O HTML não serve apenas para formatar texto. Cada tag textual possui significado semântico, importante para acessibilidade, SEO e manutenção do código.

<h1> a <h6> — Títulos

As tags de título definem a hierarquia do conteúdo. Não são usadas para tamanho visual, e sim para estrutura lógica. 

<h1>Título principal</h1>
<h2>Seção</h2>
<h3>Subseção</h3>

Boas práticas:

  • Usar apenas um <h1> por página
  • Não pular níveis (ex: h1 direto para h4)

Erro comum: usar títulos apenas para “aumentar o texto”.

<p> — Parágrafos

Representa um bloco de texto. Cria espaçamento automático antes e depois. 

<p>Este é um parágrafo.</p>

Erro comum: usar <br> repetidamente no lugar de parágrafos.

<br> — Quebra de linha

Força uma quebra de linha sem criar novo parágrafo. 

Linha 1<br>
Linha 2

Use apenas quando a quebra fizer sentido semântico.

<hr> — Divisão temática

Representa uma mudança de assunto ou seção. É semântica, não apenas visual. 

<hr>

<strong> — Importância forte

Indica que o texto tem importância semântica. Leitores de tela enfatizam essa parte. 

<strong>Atenção!</strong>

<em> — Ênfase

Dá ênfase ao texto, alterando o significado da frase. 

Eu <em>não</em> disse isso.

Diferença importante:

  • <strong> = importância
  • <em> = ênfase

<mark> — Destaque

Marca visualmente um trecho relevante. 

<mark>Texto destacado</mark>

<small> — Texto secundário

Usado para observações, direitos autorais ou notas. 

<small>© 2026</small>

<sub> e <sup> — Subscrito e sobrescrito

H<sub>2</sub>O
10<sup>2</sup>

<q> — Citação curta

Usada para citações dentro do texto. 

Ele disse: <q>HTML é semântica</q>

<blockquote> — Citação longa

Usada para citações em bloco. 

<blockquote>
  O HTML descreve o significado do conteúdo.
</blockquote>

<cite> — Fonte da citação

<cite>MDN Web Docs</cite>

<code> — Código

Representa trechos de código em linha. 

Use <code><p></code> para parágrafos.

<pre> — Texto pré-formatado

Preserva espaços, tabs e quebras de linha. 

<pre>
Linha 1
    Linha 2
</pre>

Dica: combine <pre> com <code>.

<span> — Contêiner inline

Não tem significado semântico. Usado para estilização pontual. 

<span>Texto</span>

Use com CSS, não para estruturar conteúdo.

<abbr> — Abreviações

Explica siglas e abreviações. 

<abbr title="HyperText Markup Language">HTML</abbr>

<del> e <ins> — Texto removido e inserido

<del>Antigo</del>
<ins>Novo</ins>
