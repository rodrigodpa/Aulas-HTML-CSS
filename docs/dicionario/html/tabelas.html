Tabelas no HTML

Tabelas são usadas para representar dados tabulares, ou seja, informações organizadas em linhas e colunas. Elas não devem ser usadas para layout visual.

<table> — Estrutura da tabela

A tag <table> define o contêiner principal da tabela. Dentro dela ficam linhas, células e elementos semânticos. 

<table>
</table>

<thead>, <tbody> e <tfoot>

Essas tags organizam semanticamente a tabela:

  • <thead> — cabeçalho
  • <tbody> — corpo dos dados
  • <tfoot> — rodapé
<table>
  <thead>
    <tr>
      <th>Nome</th>
      <th>Idade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Ana</td>
      <td>20</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Fim da tabela</td>
    </tr>
  </tfoot>
</table>

Boa prática: mesmo que não use <tfoot>, utilize <thead> e <tbody>.

<tr> — Linha

Define uma linha dentro da tabela. 

<tr>
  <td>Dado</td>
</tr>

<td> — Célula de dados

Representa uma célula comum da tabela. 

<td>Conteúdo</td>

<th> — Célula de cabeçalho

Usada para cabeçalhos de colunas ou linhas. Leitores de tela usam essa informação para relacionar dados corretamente. 

<th>Produto</th>

Atributo scope

Indica a qual parte da tabela o cabeçalho se refere. 

<th scope="col">Preço</th>
<th scope="row">Total</th>

<caption> — Título da tabela

Fornece um título descritivo para a tabela. É importante para acessibilidade. 

<table>
  <caption>Lista de alunos</caption>
</table>

Use sempre que possível.

Mesclagem de células

colspan

Mescla colunas. 

<td colspan="2">Total</td>

rowspan

Mescla linhas. 

<td rowspan="3">Grupo A</td>

<colgroup> e <col>

Permitem estilizar colunas inteiras. 

<table>
  <colgroup>
    <col span="1">
    <col span="1">
  </colgroup>
</table>

Muito usado em tabelas grandes.

Boas práticas em tabelas

  • Usar <th> corretamente
  • Definir scope nos cabeçalhos
  • Adicionar <caption>
  • Não usar tabelas para layout
