Estrutura do Documento HTML

Todo documento HTML segue uma estrutura básica obrigatória. Ela define como o navegador interpreta o conteúdo, garantindo compatibilidade, acessibilidade e padronização.

<!DOCTYPE>

O <!DOCTYPE html> informa ao navegador que o documento segue o padrão HTML5. Ele deve ser a primeira linha do arquivo. 

<!DOCTYPE html>

Erro comum: omitir o DOCTYPE ou usar versões antigas.

<html>

Elemento raiz do documento. Todo o conteúdo HTML deve estar dentro dele. O atributo lang define o idioma da página, essencial para acessibilidade e SEO. 

<html lang="pt-br">
</html>

Boas práticas:

  • Sempre definir lang
  • Usar códigos de idioma corretos (pt-br, en-us)

<head>

Contém informações não visíveis da página: metadados, links de estilos, scripts e título. 

<head>
  <meta charset="UTF-8">
  <title>Minha página</title>
</head>

<meta>

Define metadados sobre o documento. É uma tag sem fechamento.

charset

<meta charset="UTF-8">

viewport

<meta name="viewport" content="width=device-width, initial-scale=1.0">

SEO e informação

<meta name="description" content="Descrição da página">
<meta name="author" content="Autor">

<title>

Define o título exibido na aba do navegador e nos resultados de busca. 

<title>Título da Página</title>

Importante: cada página deve ter um título único.

<link>

Usado para vincular recursos externos, principalmente arquivos CSS. 

<link rel="stylesheet" href="style.css">

Outros usos comuns: 

<link rel="icon" href="favicon.ico">

<script>

Usado para incluir ou escrever código JavaScript. Pode ser colocado no head ou no final do body.

Script externo

<script src="script.js"></script>

Script interno

<script>
  alert("Olá!");
</script>

Boa prática: scripts no final do body.

<body>

Contém todo o conteúdo visível da página: textos, imagens, links, listas, formulários etc. 

<body>
  <h1>Conteúdo</h1>
</body>

Comentários HTML

Comentários servem para documentação e não aparecem no navegador. 

<!-- Este é um comentário -->

Atenção: comentários ainda podem ser vistos no código-fonte.
